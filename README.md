Аспирантский проект, посвященный построению методов обработки временных данных.
Основная задача - построение и визуализация результатов метода вычитания флуктуаций
при его работе с нестационарными временными рядами


Технические детали реализации:
Используется python v.: 3.9. и совместимые с ним библиотеки.
Временные ряды имеют формат DataFrame,
для их хранения и обработки используется библиотека pandas.
Покрытие тестами отдельных методов производилось с помощью библиотеки pytest.
Визуализация временных рядов производилась с помощью библиотек matplotlib и seaborn.
Интерфейс программы для работы с временными рядами реализован с помощью веб-фреймворка flask.


Структура проекта:
1) директория calc содержит основные методы генерации и преобразований временных рядов
2) директория data_raw содержит хранящиеся в текстовом формате сгенерированные временные ряды
3) директория saved_dataframes содержит DataFrame результатов преобразований временных рядов
в формате .csv
4) директория tests - тесты к программе
5) директории static и templates - компоненты для веб-интерфейса приложения


Работа с приложением
1) генерация временного ряда посредством функции или решения диффур
2) предварительная обработка - слайс, добавление шума и тд
3) ext+dfa / mwa-dfa преобразование - получение графики кривых dfa


В разработке:
1) создание и сохранение временных рядов из имеющихся данных на клиенте
2) построение результатов работы методов dfa extended или mwa-dfa без перезагрузки страницы

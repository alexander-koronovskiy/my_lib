Проект рассчета флуктуационного анализа с вычитанием среднего (аспирантская работа).

Логика проекта представляет из себя разделенную на 3 части реализацию вычисления алгоритма
DFA по временному ряду.

1 часть представляет собой запись временных рядов, генерируемых, или обрабатываемых файлами data_create в датафрейм, .csv файл;
2 часть представляет собой обработку и запись результатов обработки в датафрейм, .csv файл;
3 часть - это вывод значений из датафрейма, .csv файла; их графическое представление; реализация интерфейса

 - данные из файла
 - решение дифф-ур
